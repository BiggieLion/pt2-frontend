A lo largo de este documento se describe una propuesta de solución enfocada al desarrollo de un sistema web de análisis de riesgos crediticios. Este sistema tiene como objetivo principal el automatizar el proceso de análisis las variables contenidas en una solicitud de crédito financiero a través de una herramienta de inteligencia artificial considerando un patrón de riesgo que valide los resultados obtenidos, tomando en cuenta las garantías que ofrece el prestatario para cubrir dicho crédito, y cotejando la solicitud ingresada con solicitudes pasadas que servirán como modelo para entrenamiento de la herramienta.
